# 직장 내 괴롭힘 판단 프로세스

웹 기반 판단 도구입니다. 브라우저에서 `index.html`을 열면 바로 사용할 수 있습니다.

## 포함 파일
- `index.html` : 배포용 메인 페이지
- `직장내_괴롭힘_판단_프로세스.html` : 원본 페이지
- `괴롭힘 인지 감수성 점검(근로자용).html`
- `괴롭힘 인지 감수성 점검(사용자용).html`

## 배포
GitHub Pages 기준:
1. 저장소 생성 후 코드 푸시
2. GitHub 저장소 `Settings > Pages`
3. `Deploy from a branch` 선택
4. `Branch: main / (root)` 선택 후 저장

배포 주소 예시:
`https://<github-username>.github.io/<repo-name>/`
